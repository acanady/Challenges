## Contains Duplicate

**LeetCode quesiton 217**
**Difficulty:** Easy

Given an integer array of ```nums``` return ```true``` if any value appeares at least twice in the array, and return ```false``` if every element is distinct.

**Example 1**
```
Input: nums = [1,2,3,1]
Output: true
```
**Example 2:**
```
Input: nums = [1,2,3,4]
Output: false
```

**Example3:**
```
Input: nums = [1,1,1,3,3,4,3,2,5,2]
Output: true
```
